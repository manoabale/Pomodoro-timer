# Pomodoro-timer 🍅 

A simple and effective **Pomodoro Timer** built with **HTML**, **CSS**, and **JavaScript**.  
This timer follows the classic Pomodoro technique: **25 minutes of focus** followed by a **5‑minute break**, helping you stay productive and maintain healthy work rhythms.



## 🚀 Features

### ⏱ Core Timer Functions
- 25‑minute focus session  
- 5‑minute break session  
- Automatic switching between focus and break  
- Start, Pause, and Reset controls  

### 🎨 UI Features
- Clean and minimal interface  
- Large timer display  
- Status indicator (Focus 🍅 / Break ☕)  
- Fully responsive layout  

### 💻 Technical Features
- Lightweight vanilla JavaScript  
- Easy to customize durations  
- Works on CodePen, GitHub Pages, Netlify, and any static hosting
